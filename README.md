# DE Technology (DET) Corporate Website

A modern, professional website for Diverse Excellence Technology - a premier technology consulting firm specializing in AI, Data Analytics, and Digital Modernization.

## Overview

This website showcases DET's core philosophy of "Partnership, Not Displacement" and comprehensive service offerings across multiple sectors including Healthcare, Government, Non-Profit, K-12 Education, and Technology companies.

## Features

### Multi-Page Architecture
- **Home Page**: Hero section, services overview, partnership philosophy, client industries, and call-to-action
- **Services Page**: Detailed breakdown of all four core service offerings
- **About Us Page**: Company mission, leadership team profiles, and collaborative approach
- **Contact Page**: Contact form, company information, response times, and NAICS codes

### Design & UX
- **Professional Color Palette**: Deep Blue (#0A2540), Teal Accent (#00C49A), Cool Gray (#F6F9FC)
- **Responsive Design**: Fully responsive across all devices and screen sizes
- **Modern Animations**: Subtle hover effects, transitions, and scroll-based interactions
- **High-Quality Images**: Professional stock photos from Pexels
- **Accessibility**: Proper contrast ratios and semantic HTML structure

### Navigation
- **State-Based Routing**: Single App.tsx file with navigation managed by React state
- **Sticky Header**: Transparent on hero, becomes opaque with blur effect on scroll
- **Mobile-Friendly**: Responsive hamburger menu for mobile devices
- **Smooth Scrolling**: Animated transitions between sections

## Technical Implementation

### Built With
- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **Lucide React** for icons
- **Vite** for build tooling

### Architecture
- Single-page application with state-based navigation
- All components contained within App.tsx
- Responsive design with mobile-first approach
- Professional animations and micro-interactions

### Key Components
- `Header()` - Sticky navigation with scroll-based styling
- `HomePage()` - Landing page with hero section and service overview
- `ServicesPage()` - Detailed service offerings and capabilities
- `AboutPage()` - Company information and team profiles
- `ContactPage()` - Contact form and business information

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
